# Юзкейси: 
## 1. Створення навчальної дисципліни
* **Актор:** Викладач 
* **Передумови:** Актор авторизований як Викладач.
* **Основний сценарій:**
  1. Викладач відправляє назву нової дисципліни.
  2. Система перевіряє унікальність назви.
  3. Система зберігає дисципліну в БД.
  4. Система повертає підтвердження (HTTP 201).
* **Альтернативні сценарії / помилки:**
  * Недостатньо прав (якщо це намагається зробити Студент): HTTP 403.
  * Дисципліна з такою назвою вже існує (конфлікт): HTTP 409.

## 2. Створення лабораторної роботи
* **Актор:** Викладач 
* **Передумови:** Актор авторизований як Викладач. У базі існує хоча б одна Дисципліна.
* **Основний сценарій:**
  1. Викладач відправляє запит на створення лаби (назва, опис, ID дисципліни, дата дедлайну).
  2. Система перевіряє валідність даних (інваріант: дедлайн не може бути в минулому).
  3. Система зберігає лабу в базі.
  4. Система повертає дані створеної лаби (HTTP 201).
* **Альтернативні сценарії / помилки:**
  * Невалідні дані (наприклад, дедлайн у минулому): HTTP 400.
  * Дисципліну не знайдено: HTTP 404.

## 3. Зміна статусу лабораторної роботи
* **Актор:** Студент 
* **Передумови:** Актор авторизований як Студент. Лабораторна робота існує в системі.
* **Основний сценарій:**
  1. Студент відправляє запит на зміну статусу своєї роботи (наприклад, з "До виконання" на "На перевірці").
  2. Система перевіряє, чи є такий статус допустимим.
  3. Система оновлює запис у БД.
  4. Система повертає оновлені дані (HTTP 200).
* **Альтернативні сценарії / помилки:**
  * Лабу не знайдено: HTTP 404.
  * Передано невідомий статус або інші невалідні дані: HTTP 400.

## 4. Перегляд списку дедлайнів
* **Актор:** Студент 
* **Передумови:** Актор авторизований у системі.
* **Основний сценарій:**
  1. Студент робить запит на отримання списку лабораторних робіт.
  2. Система формує список, сортуючи його за датою дедлайну (від найближчого до найдальшого).
  3. Система повертає масив даних (HTTP 200).
* **Альтернативні сценарії / помилки:**
  * Неавторизований запит: HTTP 401.
